# 😄 7 Jokes About This Tutorial (in English)

A little comic relief for **Díl 05 — Hooks & automatizace**.

1. My `PostToolUse` hook runs `php -l` on every file Claude writes. It's the only code reviewer that's always awake and has never once said *"looks good to me."*

2. Why did the `SessionStart` hook count the PHP files out loud? Because a good pair programmer always tells you what you're getting into before you commit.

3. I asked Claude to `smaž databázi tasks.sqlite`. The `UserPromptSubmit` guard didn't stop me — it just quietly whispered *"exit 0... but are you sure about this, buddy?"*

4. Our `php-lint.sh` hook had to learn to keep quiet. Turns out shouting `php: command not found` after every single edit is not what people mean by *"continuous feedback."*

5. This whole repo is a task manager, but the real task is teaching hooks. It's the only to-do app whose most important item is *"pretend to be a to-do app."*

6. How many dependencies does this project have? Zero. The Composer walked in, saw a hand-written 26-line PSR-4 autoloader, and walked right back out.

7. The SQLite database builds its own schema on the first request and seeds itself. The first seeded task? *"Learn Claude Code."* That database is more self-motivated than most of us on a Monday.
